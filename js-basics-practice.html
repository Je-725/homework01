JavaScript 基础知识练习

一、变量的定义

JavaScript 中有三种声明变量的方式：var、let、const。

二、变量类型

JavaScript 是动态类型语言，常见类型包括：Number、String、Boolean、Null、Undefined、Object、Symbol、BigInt。

三、表达式与运算符

表达式是由操作数和运算符组成的代码片段，包括算术表达式、比较表达式、逻辑表达式等。

四、流程控制

流程控制包括条件语句（if...else、switch）和循环语句（for、while）。

五、函数

函数是执行特定任务的代码块。定义方式包括：函数声明、函数表达式、箭头函数。
